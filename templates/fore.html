About The Project

Machine learning system predicting forest cover types using environmental, topographical, and soil features

About Spruce/Fir Forests

SpruceFir forests typically grow in high-elevation, cold, and moist climates. They are dominated by Engelmann spruce and subalpine fir. These forests thrive in steep, shaded areas with dense canopies. The environment supports mosses, lichens, and shade-tolerant plants. They provide habitat for snow-dependent wildlife species. Spruce-Fir ecosystems play a major role in stabilizing mountain soil. They are highly sensitive to temperature rise and climate change. Commonly found in zones above 3000 meters.

Model Pipeline

Data Preprocessing

Raw data from Roosevelt National Forest is cleaned, missing values are handled, and categorical variables are one-hot encoded. Features are normalized using StandardScaler for consistent scaling.

Model Architecture

Random Forest Classifier with 100 estimators, max_depth=15, min_samples_split=5, and min_samples_leaf=2. The model uses Gini impurity as the splitting criterion.

Training Details

Trained on 15,120 samples (70% of data) with 5-fold cross-validation. Tested on 6,480 samples (30% of data). Training time: 45 seconds on Google Colab (GPU runtime).

Backend Integration

Flask API endpoint at http://127.0.0.1:5000/predict accepts JSON POST requests and returns predicted forest type with confidence scores.

Dataset Documentation

Roosevelt National Forest, Colorado - 30x30 meter cell data

Dataset Overview

The Forest CoverType dataset contains cartographic variables for 30x30 meter cells from Roosevelt National Forest in northern Colorado. The dataset includes 54 features and 7 forest cover type classes.

Feature Descriptions

Feature Description Data Type Range/Values
Elevation Elevation in meters above sea level Continuous 1850 - 3850
Aspect Azimuth in degrees (0-360) Continuous 0 - 360
Slope Slope in degrees Continuous 0 - 90
Horizontal Distance to Hydrology Horizontal distance to nearest surface water Continuous 0 - 1400
Vertical Distance to Hydrology Vertical distance to nearest surface water Continuous -200 - 600
Horizontal Distance to Roadways Horizontal distance to nearest roadway Continuous 0 - 7000
Hillshade 9am Hillshade index at 9am (summer solstice) Continuous 0 - 255
Hillshade Noon Hillshade index at noon (summer solstice) Continuous 0 - 255
Hillshade 3pm Hillshade index at 3pm (summer solstice) Continuous 0 - 255
Wilderness Area (4 binary) Wilderness area designation (Rawah, Neota, Comanche Peak, Cache la Poudre) Binary 0 or 1
Soil Type (40 binary) Soil type classification Binary 0 or 1

Forest Cover Type Classes

Class ID Forest Cover Type Description
1 Spruce/Fir Engelmann spruce and subalpine fir forests at high elevations
2 Lodgepole Pine Lodgepole pine dominant forests in well-drained upland areas
3 Ponderosa Pine Ponderosa pine forests in warm, dry, low-to-mid elevation terrain
4 Cottonwood/Willow Cottonwood and willow species near rivers and wet valleys
5 Aspen Quaking aspen forests that regenerate quickly after disturbances
6 Douglas-fir Douglas-fir forests in well-drained, dry, and sunny regions
7 Krummholz Stunted, wind-shaped trees at alpine treeline edges
